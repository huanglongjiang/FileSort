**中文**  
文件分类助手  
一个基于 Python 的智能文件分类工具，支持目录编辑、日志记录和自定义分类规则，帮助您快速整理电脑中的杂乱文件。 
http://www.100sucai.com/web_online_tools/FileSort.zip 

✨ 主要功能  
智能文件分类：根据文件扩展名自动分类到相应文件夹  
自定义目录结构：支持用户自定义分类规则和目标目录  
完整日志记录：详细记录所有文件操作，便于追踪和恢复  
批量处理：支持整个目录的批量整理  
配置文件管理：通过 JSON 配置文件保存分类规则  

**English**  
File Organization Assistant  
A Python-based intelligent file organization tool that supports directory editing, logging, and custom classification rules, helping you quickly tidy up messy files on your computer.  

✨ Key Features  
Smart File Classification: Automatically categorizes files into corresponding folders based on file extensions  
Custom Directory Structure: Supports user-defined classification rules and target directories  
Complete Logging: Records all file operations in detail for easy tracking and recovery  
Batch Processing: Supports batch organization of entire directories  
Configuration File Management: Saves classification rules via JSON configuration files
